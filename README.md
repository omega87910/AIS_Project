# 計畫相關內容協助執行
## 前後端實作之功能
1.登入、註冊系統之前後端設計
2.可搜尋主要、次要關鍵字取出資料庫資料
3.可新增刪除修改資料
4.可顯示長條圖比較和其他產品之資訊
5.其他後台管理系統
6.使用者名稱之顯示
7.相關操作之圖示(FontAwesome5)
8.表格排列之功能
